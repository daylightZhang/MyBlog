# 博客系统-需求文档

## 用户管理

### 用户注册

- [ ] 【基本要求】用户能根据提示完成注册

- [ ] 【基本要求】对用户设置的密码进行加密后存储，用户设置的密码不能太简单

- [ ] 【进阶要求】上传用户头像

### 用户登录

- [ ] 【基本要求】用户输入正确的账号密码可以登录
- [ ] 【进阶要求】限制用户错误密码次数
- [ ] 【进阶要求】实现免密登录

### 用户注销/登出

- [ ] 【基本要求】能够注销，登出
- [ ] 【进阶要求】清除token等信息

### 用户信息修改

- [ ] 【基本要求】修改用户的信息

### 角色管理

- [ ] 【进阶要求】实现不同角色的功能区分，权鉴
